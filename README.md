# 非24小时健身指南 (Non-24 Fitness Guide)

> 为患有非24小时睡眠-觉醒综合症 (Non-24-Hour Sleep-Wake Disorder) 的人群设计的健身训练管理方案。

## 关于非24小时睡眠-觉醒综合症

非24小时睡眠-觉醒综合症是一种昼夜节律障碍，患者的生物钟周期不是标准的24小时，导致睡眠和觉醒时间每天都会向后推移。这给传统的"每日固定时间训练"带来了巨大挑战。

## 核心理念

**忘掉墙上的时钟，只听从身体的信号。**

本方案使用 Trello 作为工具，将训练计划从"基于时间"转变为"基于状态"，让你能够灵活应对不断变化的作息时间。

## 使用方法

### Trello 看板设置

创建三个列表：

- **列表 A (未来)** - 存放待执行的训练/休息卡片
- **列表 B (现在)** - 当前周期要执行的内容
- **列表 C (历史)** - 已完成的记录

## 日常操作流程

### 🌅 情景 1：你刚睡醒

*(不管现在是凌晨3点还是下午4点)*

1. 拿起手机，打开 Trello
2. 看一眼 **列表 B (现在要做什么)**，应该是空的
3. 去 **列表 A (未来要干什么)**，按住**最上面**的那张卡片，把它拖动到 **列表 B**
4. **看一眼卡片写着什么：**
   - 🟢 **如果是"休息日"**：那这一觉醒来的十几个小时里，随便吃喝玩乐，不要去健身房
   - 🔴 **如果是"训练日"**：安排时间去练

### 🏋️ 情景 2：去健身房训练 (记录数据)

1. 点击那张"训练卡片"进入详情页
2. 在 **描述 (Description)** 或 **评论 (Comment)** 里记录今天的成绩
3. **例如写下**：`卧推 60kg x 8, 8, 8`
4. 这样以后你想"分析"时，翻回这张卡片就能看到数据

### 🛌 情景 3：你困了，准备睡觉

1. 打开 Trello
2. 把 **列表 B** 里的那张卡片，拖动到 **列表 C (历史记录)** 的最上方
3. 睡觉

## 为什么这个方法有效？

传统健身计划的问题是：它们假设你每天在同一时间醒来。但对于 Non-24 患者来说，今天8点起床，明天可能是10点，后天可能是12点...

这个方法的优势：

1. **去时间化** - 不再依赖时钟，而是依赖"睡醒"这个状态
2. **灵活性** - 无论你什么时候醒来，都能知道该做什么
3. **可追溯** - 历史记录让你能看到自己的训练模式
4. **简单** - 只需要拖动卡片，没有复杂的计算
